## 讨论内容

### 1. 用户反馈整理
用户反馈集中在以下几点：
- 搜索功能不够精确，关键词匹配度低
- 移动端体验需要优化，加载速度慢
- 希望增加夜间模式
- 导出功能支持的格式太少

### 2. 下一版本功能优先级
经讨论决定 v2.1 优先级如下：
1. 搜索算法优化（P0）
2. 移动端性能优化（P0）
3. 夜间模式（P1）
4. 导出格式扩展（P2）

### 3. 技术方案初步讨论
- 搜索：考虑引入 Elasticsearch
- 移动端：图片懒加载 + CDN
- 夜间模式：CSS 变量方案
- 导出：先支持 PDF 和 Markdown
