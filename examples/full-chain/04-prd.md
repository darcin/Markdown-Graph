# v2.1 版本 PRD

## 概述
基于用户反馈和团队讨论，v2.1 聚焦搜索体验和移动端性能优化。

## 功能需求

### F1: 搜索优化（P0）
**背景**：当前基于 SQL LIKE 的搜索精度低、响应慢
**方案**：迁移至 MeiliSearch
**验收标准**：
- 搜索响应 < 100ms
- 支持中文分词、拼音、模糊匹配
- 搜索结果按相关度排序

### F2: 移动端性能（P0）
**背景**：移动端首屏加载 >3s
**方案**：图片懒加载 + CDN
**验收标准**：
- 首屏加载 < 1.5s
- LCP < 2s

### F3: 夜间模式（P1）
**方案**：CSS 变量实现主题切换
**验收标准**：
- 支持浅色/深色/跟随系统
- 过渡动画 < 300ms

### F4: 导出格式（P2）
**方案**：扩展 PDF、Markdown 导出
**验收标准**：
- 支持 PDF、Markdown、HTML 三种格式
- 保持格式一致性
